## 二、小程序简介（用于搜索展示）

```
家庭保险管家 —— 一站式家庭保单管理工具。

【核心功能】
• 保单管理：录入全家保单，到期自动提醒
• 缺口诊断：智能分析保障缺口，查漏补缺
• 续保追踪：按时间排序，90天内到期高亮提醒
• 保费汇总：按成员/险种统计年保费，一目了然
• 数据安全：所有数据仅存手机本地，不上传服务器

【适用人群】
有多个家庭成员、多份保单需要统一管理的家庭用户。

【免责声明】
本工具为信息管理工具，不提供保险销售服务。
产品推荐仅供参考，投保前请咨询持牌保险顾问。
```

## 三、类目说明文案（审核用）

```
本小程序定位为「工具-信息查询」类目，提供以下功能：

1. 家庭保单信息录入与管理
   - 用户自行录入家庭成员及已有保单信息
   - 系统辅助整理、归类、到期提醒

2. 保障缺口分析
   - 基于用户录入的已有保单与家庭成员信息
   - 对照常见的保障类型（医疗/意外/重疾/定寿等）
   - 生成保障缺口清单供用户参考

3. 保费统计与汇总
   - 自动计算家庭年度保费支出
   - 按成员/险种分类统计

本小程序不涉及：
- 保险产品销售或代理
- 保险产品比价或排名
- 保险咨询或顾问服务
- 任何金融服务或资金交易

所有分析结果仅供用户参考，不构成购买建议。
数据完全存储在用户手机本地，服务器不接触用户数据。
```

## 五、审核注意事项

1. **不要出现以下词汇**：保证收益、承诺赔付、最佳产品、推荐购买、保险销售
2. **所有推荐类文案加"仅供参考"**
3. **确保隐私弹窗正常显示**
4. **首次提交建议用最小功能集**——不要上传太多页面，降低审核复杂度
5. **如果被驳回**：根据驳回原因修改，重新提交通常 1-2 天通过

---

## 六、审核通过后的操作

1. 发布后立即测试：用家人微信扫码确认可用
2. 观察后台数据：访问人数、页面停留
3. 收集反馈：分析页的反馈入口
4. 准备下一版：根据反馈迭代
